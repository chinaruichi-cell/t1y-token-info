Professional Profile Disclosure

The project currently uses an organization-level public representative model instead of publishing unsupported personal-profile claims. Any personal founder or team profile submitted to an explorer should be a real public profile controlled by the person named and should clearly show involvement with T1Y.

This page uses only verified project-controlled public channels and does not add unsupported personal-profile claims. Explorer reviewers should use the website, verified source code, BscScan records, public GitHub repository, and official email as the current public evidence set.
