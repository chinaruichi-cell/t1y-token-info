Public Description

T1Y is a BNB Smart Chain protocol focused on on-chain liquidity, LP-based hashrate accounting, referral rewards, node-level rewards, contract-defined burn behavior, and transparent rules around the T1Y/WBNB liquidity pool.

The project description is intentionally written from a neutral stance and does not make comparative claims such as best, fastest, most used, or most stable.

Submission Notes for Reviewers

This package is intended to help reviewers verify the project's public information without relying on private files. All links are public. The project acknowledges non-standard BEP-20 behavior, reward logic, burn behavior, and scanner notices in the security and whitepaper pages.

If an explorer requires a personal professional profile, the project should submit only a real profile controlled by the named person and updated to show T1Y involvement.
