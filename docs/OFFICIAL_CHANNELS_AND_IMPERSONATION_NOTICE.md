## Not Official Unless Later Published Here

The following should not be treated as official unless they are later added to
this page and cross-linked from the official website:

- Telegram groups
- Discord servers
- X/Twitter accounts
- Facebook pages
- Medium pages
- Reddit communities
- third-party download links
- private customer-service accounts
- screenshots shared in chats

## Security Boundaries

T1Y's public information site does not ask for:

- private keys
- mnemonic phrases
- private RPC URLs
- wallet seed backups
- remote-control access
- token approval signatures
- transfer signatures
- payment to customer-service accounts

Any request for those items should be treated as unofficial and unsafe.

## Impersonation Handling

If an explorer, wallet, scanner, or user sees a T1Y-related channel that is not
listed above, it should be treated as unverified until it is confirmed through:

1. the official website,
2. the GitHub repository,
3. the BscScan token page, or
4. the official contact email.
